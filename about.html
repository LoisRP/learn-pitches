Created by Lois Patterson.

Formats inspired by Dieter Schneider for http://www.csstemplateheaven.com.

Each MP3 file was made by creating a MIDI file with MuseScore, and then converting them using the Zamzar online converter.

The scores are cropped screenshots from MuseScore.

Although I could have done this game with single pitches, I decided to try scales to provide the listener with more context.
